# Agent útmutató

Ez a repó egy DarkOrbit-alapú webes felület, a dokumentációt lehetőség szerint magyarul tartsd naprakészen.

## Irányelvek
- PHP kódnál maradj a jelenlegi stílusnál (4 szóközös behúzás, egyszerű PSR-szerű szerkezet), és kerüld a biztonsági adatok kódba írását.
- Ha konfigurációt kell módosítanod, használj példákat vagy leírást, ne commitolj éles jelszót vagy titkot.
- Nincs automatizált teszt; ha manuálisan próbálsz valamit, írd le a futtatott parancsokat és az eredményt a válaszban.
- Dokumentáció vagy README frissítésekor magyar nyelvű, tömör leírást használj, és utalj a releváns fájlokra/mappákra.
